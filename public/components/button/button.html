Buttons

Buttons are one of the clickable elements on a web page. Buttons can be used for a wide variety of purpose.
To use a button component add class btn.

There can be different types of btn like

  • standard [btn-standard]
  • outlined [btn-outlined]
  • link [btn-link]
  • icon [btn-icon]
  • floating [btn-floating]

Each of this button type has different sub-types

  • Primary [btn-primary]
  • Secondary [btn-secondary]
  • Success [btn-success]
  • Warning [btn-warning]
  • Danger [btn-danger]
  • Disabled [btn-disabled]
