Alert

Alerts are used to attract user's attention for important information.
To use an alert component add class alert-bar, for a particular type, use the respective class alongside.

There are four types of Alerts :
1. Info alert [alert-bar-info]
2. Success alert [alert-bar-success]
3. Warning alert [alert-bar-warning]
4. Danger alert [alert-bar-danger]
