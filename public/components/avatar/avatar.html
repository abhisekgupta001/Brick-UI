Avatar

Avatars are used for displaying either user's profile picture, initials, or some abstract image
To use an avatar component add class avatar to images and avatar-initials to text elements.

There can be different types of avatars like

  • circular [avatar-circular]
  • squared [avatar-squared]
  • rounded-corner [avatar-rounded-cr]
  • outlined [avatar-outlined]
