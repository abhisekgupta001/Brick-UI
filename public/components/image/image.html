Images

Imagess are images, pictures on a web page. Images can be in form of a product image, a profile image, a banner etc.
To use an image component add class img.

There are multiple types of images on a site, some of the types are:

  • Responsive image [img-responsive]
  • rounder image[img-rounded]
  • circular image [img-circular]

Regular Responsive image

*Width and height can be changed as per need from CSS.

Rounded Responsive image

*Width and height can be changed as per need from CSS.

Circular Responsive image

*Width and height for a circular image has to be same.
