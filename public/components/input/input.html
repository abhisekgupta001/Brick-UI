Inputs

Inputs are elements on UI that helps a used to enter values.
To use an input component add class input-field or input-btn.

There are multiple types of input, some of the types are:

  • Input Text Field [input-field-text]
  • Input Number Field [input-field-number]
  • Input Password [input-field-password]
  • Input Email [input-field-email]
  • Input Radio Button [input-btn-radio]
  • Input Check Box [input-btn-checkbox]
  • Input Textarea [input-field-textarea]
